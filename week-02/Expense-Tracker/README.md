# Expense Tracker Application

A simple **Expense Tracker** built with **React** as a learning project during **Week 1 & Week 2** of studying modern front-end development.

The application allows users to add expenses and view them in a list, focusing on understanding **React fundamentals** such as components, props, state, and form handling.

The project runs locally using **Vite**.
